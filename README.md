# Recipe Book App - In Progress
(https://myrecipe-book-app.netlify.app/)
A vintage-themed recipe web app built with React, JavaScript, HTML, and CSS. Users can add, edit, delete, and search recipe cards.

## Features
- Add new recipe cards
- Edit existing recipes
- Delete recipes
- Search recipes alphabetically
- Vintage-inspired card layout and UI styling

## Tech Stack
- React
- JavaScript
- HTML
- CSS

## What I Learned
- Managing component state
- Building CRUD functionality
- Creating reusable UI components
- Styling a themed user interface
